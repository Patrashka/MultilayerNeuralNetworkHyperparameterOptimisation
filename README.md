# MultilayerNeuralNetworkHyperparameterOptimisation

# Tarea 6 - Red Neuronal Multicapa con Optimización de Hiperparámetros

Repositorio para la Tarea 6 de la materia **Inteligencia Artificial 2**. Este proyecto implementa una red neuronal profunda para clasificar tipos de vidrio basándose en su composición química, aplicando optimización de hiperparámetros y *early stopping*.

## Autores
* Adrián Marcelo Garza Morton
* Patricio Dávila Santos

## Archivos del Proyecto
* `IA2_Tarea6.ipynb`: Libreta de Jupyter con el código fuente, entrenamiento del modelo y análisis de resultados.
* `glass.data` / `glass.names`: Dataset original de identificación de vidrios.
* `modelo_vidrio_mlp.keras`: Modelo final entrenado y exportado.
